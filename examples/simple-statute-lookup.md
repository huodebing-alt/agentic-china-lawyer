# 示例 1（Simple）— 单条法规查询

## 场景

律师在写一份家事案件代理意见，想确认民法典关于夫妻共同财产范围的条款是怎么说的。

## 用户输入

```
帮我查一下民法典关于夫妻共同财产范围的条款。
```

## task-router 判断

| 维度 | 取值 | 分 |
| --- | --- | --- |
| 长度 | 22 字 | 0 |
| 领域数 | 1（家事） | 0 |
| 文档 / 主体 | 0 | 0 |
| 输出预估 | ~ 500 字 | 0 |
| 并行子目标 | 0 | 0 |

总分 **0 → Simple**。直接 dispatch。

## 编排

```
task-router
  → research-specialist-statute
    → mcp__statutes-rag__search("夫妻共同财产")
  → task-router 收回 → 用户
```

无需 aggregator，单 specialist 已输出完整答案。

## 期望输出（摘要）

```
# 民法典关于夫妻共同财产范围

## 核心条款
- 《中华人民共和国民法典》第 1062 条
  （statutes-rag MCP 返回的原文，律师请以官方为准）

## 相关条款
- 第 1063 条（个人财产范围）
- 第 1065 条（约定财产制）

## 注意
- 婚前 vs 婚后所得归属
- 知识产权收益 / 投资收益的归属判定
- 一方个人财产与共同财产的转化

## 司法解释
- 最高人民法院关于适用《民法典》婚姻家庭编的解释（一）

⚠️ 引用以官方原文为准，律师执业请通过 flk.npc.gov.cn 或付费数据库核对。

> 本文件由 Claude 辅助生成，仅供执业律师内部参考。
> 最终对外签字、盖章、提交前，必须由律师亲自复核。
```

## 耗时与 token

- 1 次 router 判断（≤ 500 token）
- 1 次 specialist + 1 次 MCP 调用（≤ 2000 token）
- 无需 aggregator / checker
- 总 token < 3000，秒级返回
