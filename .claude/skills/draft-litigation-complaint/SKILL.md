# /draft-litigation-complaint

## 反问
1. 原告 / 被告信息
2. 诉讼请求（不超过 N 项，金额精确到元）
3. 事实与理由（陈述时间线）
4. 证据清单
5. 管辖法院 + 管辖依据
6. 适用法律

## 路由
`litigation-prep-specialist` → `drafting-stylist`

## 输出
```
                     民事起诉状

  原告：……  住所：……  联系电话：……
  法定代表人 / 负责人：……
  委托诉讼代理人：……（律师事务所、律师）

  被告：……  住所：……  联系电话：……

  诉讼请求：
    一、判令被告 ……
    二、判令被告 ……
    三、判令被告承担本案诉讼费用

  事实与理由：
    一、<事实陈述>
    二、<法律分析>

  此致
  <法院全称>

                                  起诉人：（签字 / 盖章）
                                  YYYY 年 MM 月 DD 日

  证据清单：
    1. ……
    2. ……
```

## 红线
- 管辖必须查
- 诉讼时效必须查
- 诉请金额对应案件受理费
