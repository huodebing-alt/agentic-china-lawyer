# /calculate-pit

## 输入

- 收入类型：工资 / 劳务 / 稿酬 / 特许权使用费 / 经营 / 利息股息 / 财产租赁 / 财产转让 / 偶然
- 金额
- 专项扣除（社保 + 公积金）
- 专项附加扣除（子女教育 / 继续教育 / 大病医疗 / 住房贷款利息 / 住房租金 / 赡养老人 / 3 岁以下婴幼儿照护）
- 居民 / 非居民

## 路由

`tax-specialist`

## 输出

```
# 个税测算 — <场景>

## 一、假设条件
## 二、应纳税所得额计算
## 三、税率与速算扣除数
## 四、应纳税额
## 五、已预扣预缴
## 六、应补 / 应退
## 七、税收筹划建议（合法）
⚠️ 测算仅供参考，最终以税务机关核定为准。
```
