# /draft-equity-incentive

## 输入
- 公司形态（有限 / 股份 / 上市筹备）
- 激励池比例（一般 10-15%）
- 激励对象（高管 / 核心员工 / 顾问）
- 工具（期权 / RSU / 虚拟股 / 分红权）
- 归属期 / 行权价 / 退出机制

## 路由
`corporate-specialist` + `tax-specialist` 并行（个税适用财税 [2016] 101 号 / [2018] 164 号）

## 输出
- 激励池方案 + 工具选择理由
- 授予协议范本
- 行权 / 归属时间表
- 税务测算
- 退出机制（离职、IPO、回购）
