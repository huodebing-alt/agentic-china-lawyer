# /review-shareholder-agreement

## checklist
- 出资形式与到位时间
- 股权比例与表决权安排
- 董事会 / 监事会组成
- 重大事项一致同意 / 多数决
- 反稀释（完全棘轮 / 加权平均）
- 优先购买、共同出售、拖售
- 知情权 / 检查权
- 退出（IPO / M&A / 回购）
- 创始人 vesting / 不竞争
- 争议解决（仲裁优先）
