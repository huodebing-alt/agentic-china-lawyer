# /draft-NDA-cn

## 反问

1. 双方名称
2. 单向 / 双向 / 多方
3. 保密信息范围（技术 / 商业 / 客户 / 财务 / 数据）
4. 保密期限（一般 3-5 年，超长可达 10 年）
5. 例外情形（公开 / 独立开发 / 强制披露）
6. 违约金 / 损害赔偿
7. 管辖 / 仲裁

## 输出

标准模板（略，见模板库）。

## 红线

- 不能约定保密期限"永远"（违反"合理期限"原则）
- 强制披露条款必须有"告知守约方"义务
