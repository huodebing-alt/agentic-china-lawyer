# /draft-lvshihan

## 调用

```
/draft-lvshihan
```

会反问以下要素：

1. 委托人 / 客户全称
2. 对方全称（自然人 / 公司）
3. 事由（200 字以内）
4. 主张的法律依据（律师可指定，否则由 specialist 检索）
5. 诉求（停止侵权 / 支付款项 / 履行义务 / ……）
6. 期限（要求对方在 X 日内回复 / 履行）
7. 律师事务所信息 + 经办律师 + 执业证号

## 路由

`contract-specialist` + `litigation-prep-specialist` 协作 → `drafting-stylist`

## 输出模板

```
                      律师函

                                函字 [YYYY] 第 X 号

致：<对方全称 / 地址>

  本所接受 <委托人全称> 委托，就 <事由> 一事，特郑重函告如下：

  一、<事实陈述>
  二、<法律分析>（引《XX 法》第 X 条 等）
  三、<对方违法 / 违约的具体表现>
  四、<本所基于上述事实和法律，特此要求：>
        1. <诉求 1>
        2. <诉求 2>
        ……
  五、<期限>：望贵方于本函送达之日起 X 日内 ……
        逾期 ……，本所将 ……

  特此函达。

                           <律师事务所名称>（盖章）
                           经办律师：<姓名>
                           执业证号：<XXXXX>
                           YYYY 年 MM 月 DD 日

  附：<授权委托书 / 证据材料清单 / 等>
```

末尾必带项目免责语。

## 风格

- 措辞克制、不带攻击性形容词
- 不替客户做超出授权的承诺
- 不威胁刑事报案（除非已有事实基础）
