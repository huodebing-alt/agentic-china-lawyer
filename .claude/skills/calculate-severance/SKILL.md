# /calculate-severance

## 输入
- 城市（决定社平工资）
- 入职日期 / 解除日期
- 月工资（基本 + 奖金 + 津贴的平均）
- 解除事由

## 路由
`labor-specialist`

## 公式
```
工作年限 = 解除日期 - 入职日期
经济补偿金 = MIN(月工资, 当地社平工资×3) × 工作年限（不满半年=0.5，满半年不满 1 年=1）
违法解除赔偿金 = 经济补偿金 × 2
```

## 输出
- 工作年限精确到月
- 基数 + 是否封顶
- 补偿金 / 赔偿金金额
- 适用法条
