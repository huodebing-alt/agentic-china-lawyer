# /check-data-export

## 输入
- 数据类型（重要数据 / 个人信息 / 敏感个人信息）
- 数量级
- 接收方所在国 / 地区
- 出境目的

## 路由
`data-protection-specialist` + `cross-border-specialist`

## 输出
- 适用路径（评估 / 标准合同 / 认证）
- 是否在豁免清单（《促进和规范数据跨境流动规定》）
- 自由贸易港 / 临港 / 横琴等地特殊安排
- 标准合同范本 + 备案流程
- PIA 报告要点
