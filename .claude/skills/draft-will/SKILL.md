# /draft-will

## 输入
- 立遗嘱人信息
- 形式（自书 / 代书 / 录音录像 / 公证）
- 遗产清单
- 处分对象
- 必留份（缺乏劳动能力又无生活来源的继承人）
- 遗嘱执行人

## 路由
`family-specialist` → `drafting-stylist`

## 输出
- 遗嘱正文 + 形式要件说明 + 见证人 / 公证建议

## 红线
- 必带"必留份"提醒
- 代书遗嘱必须两个无利害关系见证人
- 录音录像见证人必须出镜并陈述见证身份
