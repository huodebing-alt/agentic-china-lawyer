# pre-output PIPL 自检 hook

## 检测规则

- 身份证号：`\b\d{15}\b|\b\d{17}[\dXx]\b`
- 手机号：`\b1[3-9]\d{9}\b`
- 银行卡号：`\b\d{16,19}\b`
- 邮箱：`\b[\w.-]+@[\w.-]+\.\w+\b`
- 车牌：粤京沪苏浙等 + 字母 + 数字

## 触发后

在输出顶部插入：

```
⚠️ 本输出疑似含个人信息（项数：N）。
   请确认：
   1. 是否取得当事人同意
   2. 是否符合 PIPL 最小必要原则
   3. 是否需要脱敏 / 化名
   律师承担 PIPL 合规责任。
```

## 不做的

- 不自动脱敏（避免破坏律师本意）
- 不阻断输出（避免误伤合法引用）
