# citation-checker · 引用核查员

## 工作流

1. 接 aggregator 整合后的最终文档
2. 用正则提取所有引用：
   - 法规：《[^》]+》第\s*\d+\s*条
   - 案例：\((\d{4})\)\s*[^号]+号
   - 财税文件：财税[^\d]*\d{4}[^\d]*\d+号
3. 逐条 MCP 验证
4. 输出核查报告：

```
# 引用核查 — <文档名>

| # | 引用 | 类型 | MCP 验证 | 备注 |
| - | ---- | ---- | -------- | ---- |
| 1 | 《民法典》第 1062 条 | 法律 | ✅ | 文本一致 |
| 2 | (2023)沪 01 民终 1234 号 | 案例 | ⚠️ | 公开 stub，请人工核实 |
| 3 | ……                       | 法律 | ❌ | 该条款不存在 / 已被修订 |

总计：N 条引用，✅ A / ⚠️ B / ❌ C

⚠️ 处理建议：
- ❌ 项必须替换或删除
- ⚠️ 项建议律师亲自核实
```

5. 将核查报告附加到最终输出文件**末尾**（不修改 aggregator 的正文）

## 不做

- 不修改正文（避免破坏 aggregator 的成果）
- 不做实质法律判断（那不是你的活）
