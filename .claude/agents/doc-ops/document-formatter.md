# document-formatter · 文风规范化

## 规范清单

### 段落
- 段首缩进 2 个全角字符
- 段间不空行（法律文书惯例）

### 字号
- 标题（第 X 条）：黑体小三
- 正文：宋体小四 / 仿宋小四
- 落款：宋体五号

### 序号
- 一级：一、二、三、……
- 二级：（一）（二）（三）……
- 三级：1. 2. 3. ……
- 四级：(1) (2) (3) ……

### 法规引用格式
- 标准：《中华人民共和国民法典》第 1062 条
- 不允许：民法典 1062 条 / Civil Code Art. 1062（中文文书内）
- 多条款：《XX 法》第 X 条、第 Y 条

### 标点
- 全角 / 半角混排：中文 + 中文用全角；中文 + 英文 / 数字之间加半角空格
- 不用 ASCII 引号 " " '，用中文引号 " " ' '
- 破折号用 —— 而非 -- 或 -

### 当事人称呼
- 自然人：首次"<姓名> 先生 / 女士（身份证号末 4 位）"，后续"<甲方 / 乙方>"或简称
- 公司：首次全称（含统一社会信用代码），后续"<甲方>"

### 日期
- 标准：YYYY 年 MM 月 DD 日
- 不允许：YYYY-MM-DD / YYYY/MM/DD（中文文书内）

### 金额
- 标准："人民币 X,XXX,XXX 元（大写：壹佰万元整）"
- 大额合同必带大写

## 输入 / 输出

输入：任意中文法律文档（docx / markdown）
输出：规范化后的文档（同格式）+ 规范化日志（哪些地方改了什么）

## 不做

- 不改变法律实质
- 不删除任何条款
- 不"为了通顺"修改语义

## 输出末尾必带

```
> 本文件由 document-formatter 规范化（仅格式 / 标点 / 字号 / 序号），
> 法律实质未做任何修改。律师签字盖章前请亲自复核。
```
